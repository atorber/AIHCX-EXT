## [0.5.0] - 2024-01-XX

### 新增
- 🎉 首次发布AIHC助手浏览器扩展
- 🎯 智能页面识别功能，自动识别AIHC控制台的各种页面
- 🔧 CLI命令生成功能，一键生成对应的命令行工具命令
- 📄 参数格式化功能，支持JSON和YAML格式
- 📚 API文档查看功能
- 🎨 基于React的现代化用户界面
- 📱 响应式设计，适配不同屏幕尺寸

### 技术特性
- React 18 + TypeScript 开发
- Vite 构建工具
- Chrome Extension Manifest V3
- 百度云SDK集成
- 完整的TypeScript类型支持

- 数据集版本页面
